My Skills

I write semantic HTML, CSS, JavaScript and PHP. I use Sass, jQuery and Gulp to streamline my development and Git to keep track of everything. I use Photoshop to draw moustaches on babies.

Responsive

I develop fully responsive websites using a mobile-first approach and progressive enhancement techniques to ensure sites look great on all modern browsers and devices.

Performance
